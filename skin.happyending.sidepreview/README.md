# HappyEnding Preview Skin

Experimental Kodi 21 Omega skin focused on video add-on browsing.

Main video layout:
- List on the left.
- Large focused thumbnail/poster on the right.
- Minimal info under the image.
- Extra view modes: side list, poster wall, landscape wall, compact list.

Important:
This is a lightweight test skin, not a full Estuary replacement. Keep Estuary installed so you can switch back if needed.

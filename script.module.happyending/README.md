# HappyEnding Tools

Helper module for a patched Kodi video add-on.

It provides:
- Custom length filter: minimum and maximum minutes.
- Long-first / short-first sorting before Kodi displays the list.
- Thumbnail fallback: thumb, poster, icon and fanart all point to the best available image.
- HappyEnding SidePreview view auto-switch.

This module does **not** bypass providers, unblock sites, remove restrictions, or fetch media by itself.
